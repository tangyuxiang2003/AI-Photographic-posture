# 图标说明

请在此目录下添加以下图标文件：

## 必需图标

1. **search.png** - 搜索图标（40x40rpx）
2. **local.png** - 本地Pose图标（48x48rpx）
3. **close.png** - 关闭图标（32x32rpx）

## 临时方案

在正式图标准备好之前，可以：
1. 使用微信小程序自带的图标
2. 或者暂时注释掉图标相关代码
3. 使用 text 文本代替图标

## 占位图片

**placeholder.png** - 姿势占位图（建议尺寸：320x420rpx）
请将占位图片放在 `miniprogram/images/` 目录下
