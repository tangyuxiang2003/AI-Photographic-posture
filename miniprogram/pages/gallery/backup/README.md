# 姿势厅页面备份说明

## 备份时间
2025年11月11日 15:00

## 修改内容

### 新增功能
1. **搜索框**: 顶部添加搜索框,支持搜索场景、人物或物品
2. **标签筛选**: 添加10个标签词(浪漫满屋、不露脸、情侣、闺蜜、自拍、鲜花、古灵精怪、创意照、超市、合照)
3. **本地 Pose 分区**: 添加"本地 Pose"标题区域
4. **使用按钮**: 每个姿势卡片上添加"使用"按钮

### 删除内容
1. 删除原有的渐变色头部(标题和副标题)
2. 删除原有的分类标签(推荐、人像、风景、创意)
3. 删除"使用本地 Pose"弹窗提示
4. 删除底部蓝色条及其内容

### 样式调整
1. 搜索框采用圆角设计,背景色为浅灰色
2. 标签词采用圆角胶囊样式,选中时背景变黑色
3. 姿势卡片添加渐变遮罩和黄色"使用"按钮
4. 整体背景色跟随主题色

## 文件列表
- `index.wxml` - 页面结构
- `index.wxss` - 页面样式
- `index.js` - 页面逻辑
- `index.json` - 页面配置(未修改)

如需恢复原始版本,请将此备份文件夹中的文件复制回上级目录。
